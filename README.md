# VenueWala

A wedding and event venue discovery app built with React.

## Features

- Browse and filter venues across Mumbai, Delhi, Bangalore, Jaipur, and Hyderabad
- Wishlist, booking simulation, and user auth (localStorage-based)
- Dark mode and English/Hindi language toggle
- 12 pages including map view, profile, and booking confirmation

## Tech Stack

- Vite + React 19 + React Router v7
- Plain CSS with CSS variables (no UI library)
- No backend — all data persisted in localStorage

## Getting Started

```bash
npm install
npm run dev
```

Opens at `http://localhost:5173`

## Build

```bash
npm run build
```
